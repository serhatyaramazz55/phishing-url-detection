# Phishing URL Detection

Machine learning based phishing website detection system.

## Project Overview

This project aims to detect phishing websites by analyzing URL and domain characteristics. The system extracts multiple features from a given URL and predicts whether it is legitimate or phishing using machine learning models.

## Technologies

- Python
- Flask
- LightGBM
- Pandas
- NumPy
- BeautifulSoup
- Requests
- WHOIS

## Features

- URL feature extraction
- Domain analysis
- Website inspection
- Phishing prediction
- REST API integration

## Extracted Features

- Domain Age
- Google Index
- URL Length
- Number of Digits
- IP Address Usage
- WWW Count
- Phishing Hints

## Model

- LightGBM Classifier

## Future Improvements
